Como funciona o coração?

O coração é um órgão que funciona como uma bomba que contrai e relaxa, levando sangue para todo o corpo. O sangue do corpo vai para o coração e dele chega aos pulmões, onde recebe oxigênio. Dos pulmões, o sangue volta para o coração e, em seguida, é distribuído para todo o corpo. Para entender melhor como funciona o seu coração, assista ao vídeo abaixo.
